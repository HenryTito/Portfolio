Portfolio

This is my personal portfolio project built with **React** and **Vite**, focused on performance, modern UI, and smooth animations.

The goal of this project is to present my skills, projects, and experience as a software developer in a clean and professional way.

[Check it out!](https://henrytito.com.br/)


---

##  Technologies

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Parallax Tilt
- Three.js

---

##  Installation

Clone the repository:

git clone https://github.com/HenryTito/Portfolio.git

npm install


npm run dev
